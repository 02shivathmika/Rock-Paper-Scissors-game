✂️ Rock Paper Scissors Game   

A fun and interactive web-based Rock Paper Scissors game built with HTML, CSS and JavaScript where you play the classic game against the computer!

🎮 How to Play  
1) Click on Rock, Paper, or Scissors
2) The computer randomly picks its choice
3) The winner is decided based on the classic rules:
   
    🪨 Rock beats ✂️ Scissors
   
    📄 Paper beats 🪨 Rock
   
    ✂️ Scissors beats 📄 Paper

Tie if both choose the same!

🛠️ Built With
1) HTML
2) CSS
3) JS


🚀 How to Run  

Just open ex1.html in your browser!
